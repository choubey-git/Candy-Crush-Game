# 🍬 Candy Crush Game

A simple and interactive Candy Crush style game built using **HTML**, **CSS**, and **JavaScript**.  
This project demonstrates drag-and-drop functionality, DOM manipulation, game logic implementation, and responsive UI design using vanilla JavaScript.

---

## 🚀 Features

- 🎮 Interactive Candy Crush gameplay
- 🖱️ Drag and drop support
- 🎨 Attractive UI with CSS styling
- 🧠 Match detection logic
- 📈 Score tracking system
- 🔄 Automatic candy replacement
- ⚡ Built using pure JavaScript (No frameworks)

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
